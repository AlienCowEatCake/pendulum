User manual

Launching the program the user see the main window. The user interface consists of:

1. Main menu. It is responsible for access to graphs and help information.

2. Main graphic window. There was placed the model of pendulum. There is a possibility to zoom in, zoom out, rotate the model around an arbitrary axis with the mouse.

3. Slider to adjust the mass. Allows you to set the weight in the range from 0.01 to 0.5 kg

4. Slider to adjust the initial displacement. Allows you to set the initial displacement in the range from -0,8 to 0,8 m.

5. Slider to adjust the spring constant. Allows you to set the spring constant in the range from 25 to 250 N/m.

6. Slider to adjust the damping coefficient. Allows you to set the damping coefficient in the range from 0 to 9,99 kg/s.

7. Stopwatch. Displays the time from the start of experiment.

8. Slider to adjust the speed of experiment. Allows to speed up and slow down the experiment.

9. Slider to adjust graphic options. At better quality the motion of the pendulum will be smoother, but on it may slow the speed of the experiment on low-performance computers.

10. Button “Start/Pause/Resume”. This button is used to start and pause the execution of the experiment. After the start of experiment blocked elements 3-6 of interface are blocking.

11. Button “Stop”. Interrupts the execution of the experiment, unlocks all elements of the interface and allows to input another initial data.

Opening the graph user see the window like this:

1. Dependent variable.

2. Scale tag.

3. The curve showing the dependence of the selected parameter of the system as a function of time.

4. Time.

5. Scale tag.
